# Qdrant Edge

This is a home for the [`qdrant-edge`](https://crates.io/crates/qdrant-edge)-related code.

The source code for the `qdrant-edge` package is autogenerated by the [amalgamate.py](amalgamate.py) script and placed in `./qdrant-edge/` in this directory (gitignored).

If you need to make changes in the `qdrant-edge` package, edit the original packages in this repo ([`/lib`](/lib) in the repo root).

## How run and test

```bash
./amalgamate.py
./examples/prepare_facet_snapshot.sh

cargo check -p examples
cargo run -p examples --bin demo
cargo run -p examples --bin …
```

(see full binary list in [examples/src/bin](examples/src/bin))

## How to publish to crates.io

1. Update the `VERSION` in [amalgamate.py](amalgamate.py).
2. Run [Qdrant Edge Rust Release](https://github.com/qdrant/qdrant/actions/workflows/edge-rust-release.yml) workflow.
