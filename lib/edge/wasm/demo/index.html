Qdrant WASM

Data persists in OPFS.

1. Initialize

import { QdrantWasm } from './qdrant-wasm.js';
const qdrant = new QdrantWasm();
await qdrant.initialize(); // This downloads + caches the Qdrant WASM binary (~18MB). Takes a second or so.

2. Prepare Shard

await qdrant.prepareShard(); // The config is hardcoded for the demo

3. Upsert a Point

await qdrant.upsert(1, [0.18, 0.36, 0.54, 0.73], { label: "alpha", value: 1 });

4. Retrieve by ID

const points = await qdrant.retrieve([1]);

5. Search

const results = await qdrant.query([0.18, 0.36, 0.54, 0.73], 3);

6. Count Points

const count = await qdrant.count();

7. Scroll

const page = await qdrant.scroll({ limit: 5 });

Cleanup
